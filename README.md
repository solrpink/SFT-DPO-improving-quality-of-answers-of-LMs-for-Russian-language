# SFT-DPO-improving-quality-of-answers-of-LMs-for-Russian-language
Проект посвящен файн-тьюнингу модели gemma3-4b-it на синтетических данных на основе таксономии Блума. Таксономия Блума — это классификация учебных целей, которая разделяет мышление на 6 уровней сложности. Она помогает педагогам и методистам выстраивать программу от простого к сложному — от базового запоминания фактов до создания новых идей и критического анализа. Идея датасета, обучить модель на данных в таком порядке, как дети изучают в рамках школьной программы.

# Получение данных
Данные для SFT/DPO были сгенерированы моделью qwen3-32b. Сам процесс генерации был разделен на три ключевых этапа:

1. Извлечение концептов (и их нормализация и дедубликация). Сами концепты извлекались из российских учебных планов с 5 по 11 классы доменов: химия, биология, физика, география (основной цикл предметов естественных наук в школе). 
Далее пример, из части датасета по химии:
```json
{
  "10 класс": [
    "кислотные свойства",

    "муравьиная кислота",
    "производные карбоновых кислот",
```
